2. 数组中重复的数字

  • 题目: 在一个长度为n的数组里所有数字都在0-n-1的范围内，数组中有某些数字是重复的，但不知道有几个数字重复了。

  • 解决问题的关键在于数字的下标要和数字本身对齐，进行遍历，如果第一个数不是0，就和第一个数数值的下标的数进行互换，这样能确保后面那个数的下标和数值对应，当出现对换的那个数的下标等于其数值时，说明有重复发生

  • 题目变型:不修改数组找出重复的数字，在一个长度为n+1的数组里的所有数字都在1~n的范围内，所以数组中至少有一个数字是重复的，请找出数组中任意一个重复的数字，但不改变数组

  • 如果能创建一个辅助的数组，就可以把数组复制到辅助数组，思路就是将1到n的数字从中间的数字分成两半，统计出现的次数，然后统计这几个数字在数组中出现的次数是不是都是一次，如果不是说明，这里面有重复数字，继续二分递归，直到当只剩一个数字时，其数量大于1，然后返回这个数字

4. 替换空格

  • 题目：实现一个函数，把字符串中每个空格替换成’%20’
  • 第一种方法就是用Python的map函数，迭代字符串，将字符串中的空格替换为%20。第二种方法就是先根据空格的个数扩充字符串，用第一个指针指向字符串现有值的尾部，第二个指针指向字符串的尾部，然后从后往前遍历，将第一个指针的值赋给第二个指针指向的值，当遇到空格时，第二个指针分别赋值% 2 0，然后指针往前移动，直到第一个指针遍历到字符串的头部停止。
