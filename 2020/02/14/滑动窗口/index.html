滑动窗口

发表于 2020-02-14 | 分类于 算法与数据结构

解决求一些关于子串的问题，求在A中符合对B一些限定规则的子串或者对A一些限定规则的结果

1.题型
  • 已知条件
    • 目标串A
    • 子串B
  • 问题
    • 求在A中符合对B一些限定规则的子串或者对A一些限定规则的结果
2.思路

对于这种题目，使用双指针技巧，双指针在数组链表等数据结构中使用很频繁，

  • 定义一个双指针表示一个滑动窗口的左边和右边
    • left=0 right=0
  • 定义一个窗口，一个限定条件
    • window
    • needs
  • 先进行扩大窗口也就是移动right指针，将值加入窗口中，然后再移动left指针缩小left指针，然后再继续遍历，直到right指针到达尾部。
3.伪代码
4.实例

leetcode 209

给定一个含有 n 个正整数的数组和一个正整数 s ，找出该数组中满足其和 ≥ s 的长度最小的连续子数组。如果不存在符合条件的连续子数组，返回 0
